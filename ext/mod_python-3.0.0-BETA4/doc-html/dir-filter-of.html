5.2.2 PythonOutputFilter

Syntax: PythonOutputFilter handler name
Context: server config
Module: mod_python.c

Registers an output filter handler under name name. Handler is a module name optionally followed :: and a callable object name. If callable object name is omited, it will default to "outputfilter". Name is the name under which the filter is registered, by convention filter names are usually in all caps.

To activate the filter, use the AddOutputFilter directive.
